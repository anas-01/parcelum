--------------
### START APPLICATION
--------------

To start the application open a terminal in this folder and type:

``` bash
$ yarn install
$ yarn start
```

Go to http://localhost:3000

Login using:

username:   admin
password:   password


--------------
### CONFIGURE
--------------

Set properties in:
* For local/dev environments `config/properties.dev.ts`
* for production environments `config/properties.prod.ts`
